## ✨ Features

🔍 **Instant Word Search**  
- Get meanings, phonetics, and usage instantly  

🔐 **Secure Authentication**  
- Login & Signup using Firebase  

🎮 **Interactive Word Game**  
- Learn with hints, answers & gameplay  

🎨 **Beautiful UI + Animations**  
- Smooth effects (petals 🌸 / stars ✨)  

📱 **Fully Responsive**  
- Works on mobile & desktop
